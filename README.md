# Sitra
#### American Sign Language to Speech Translator using image processing and neural networks


## Required modules

###### keras
###### tensorflow
###### numpy 
###### open-cv
###### playsound

## How to use

#### clone the repository
git clone https://github.com/midhun-b-m/SItra

#### open final2.0 folder 
cd Sitra/final2.0

#### Run the python file 
python 'final2.0.py'
